# Notas de revisão humana

## Objetivo

Registrar avaliação humana dos alertas.

## Campos recomendados

- alerta analisado
- revisor
- data
- conclusão
- justificativa
- necessidade de aprofundamento
- documentos adicionais necessários
- classificação final

## Classificações possíveis

- relevante para aprofundamento
- falso positivo provável
- inconclusivo
- depende de documento textual
