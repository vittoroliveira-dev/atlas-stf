# Hipóteses de pesquisa

## Objetivo

Registrar hipóteses analíticas sem tratá-las como fatos.

## Estrutura de uma hipótese

- identificador
- formulação
- motivação
- dados necessários
- baseline necessário
- risco de falso positivo
- estratégia de validação
- status

## Exemplo de formulação adequada

"Em subconjuntos comparáveis da classe X, no período Y, o ministro Z pode apresentar taxa atípica do andamento W."

## Exemplo de formulação inadequada

"O ministro Z favoreceu o grupo A."
