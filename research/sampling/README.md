# Amostragem

## Objetivo

Documentar estratégias de amostragem usadas para validação, revisão manual e testes metodológicos.

## Finalidades

- montar amostras de processos para revisão humana;
- testar qualidade de grupos comparáveis;
- avaliar alertas;
- comparar recortes temporais.

## Regras

1. Toda amostra deve registrar critério de seleção.
2. Toda amostra deve registrar objetivo.
3. Toda amostra deve registrar limitações.
4. Amostras exploratórias e amostras de validação não devem ser confundidas.

## Registro mínimo

- identificador da amostra
- objetivo
- universo de origem
- critério de seleção
- tamanho
- limitações
- responsável
- data
