# Raw — eventos externos

## Objetivo

Armazenar registros brutos de fatos externos documentados, usados apenas como contexto temporal complementar.

## Uso esperado

- comparação antes/depois;
- contextualização de mudanças de padrão;
- suporte a hipóteses temporais.

## Restrições

- não usar para inferir causalidade automática;
- não misturar com a camada principal de decisões;
- sempre registrar fonte e data.

## Registro mínimo

- título do evento
- data
- fonte
- resumo
- grau de confiança editorial
