# Raw — transparência

## Objetivo

Armazenar exportações brutas provenientes do portal de transparência do STF, especialmente as relativas a Registro e Distribuição.

## Uso no projeto

Esta é a principal fonte estruturada do corpus atual para:
- processos;
- fatos decisórios;
- partes;
- advogados;
- ministros;
- assuntos;
- séries temporais.

## Regras

- preservar nome original do arquivo;
- registrar data da exportação;
- registrar filtro aplicado, quando conhecido;
- registrar duplicidades identificadas;
- nunca sobrescrever a exportação bruta.

## Registro mínimo por arquivo

- nome do arquivo
- data de coleta
- origem
- hash
- observações de cobertura
- observações de integridade
