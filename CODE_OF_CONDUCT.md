# Código de Conduta

## Compromisso

Este projeto busca um ambiente de colaboração respeitoso, objetivo e tecnicamente rigoroso. Discussões devem priorizar precisão, auditabilidade e boa-fé.

## Comportamentos esperados

- criticar ideias e artefatos, não pessoas;
- registrar discordâncias com base técnica verificável;
- respeitar limites metodológicos do projeto;
- sinalizar incertezas com clareza;
- manter linguagem profissional e não acusatória.

## Comportamentos inaceitáveis

- assédio, intimidação ou ataques pessoais;
- uso do repositório para acusações não sustentadas;
- distorção deliberada de dados, fontes ou contexto;
- publicação de material sensível de forma irresponsável;
- insistência em conclusões incompatíveis com a metodologia documentada.

## Aplicação

Mantenedores podem editar, restringir ou remover contribuições que violem este código de conduta ou prejudiquem a integridade metodológica do projeto.
