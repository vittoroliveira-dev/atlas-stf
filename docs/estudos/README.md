# Estudos de Referência

Estudos empíricos selecionados que fundamentam a metodologia analítica do Atlas STF.
Mapeados contra 4 perguntas de pesquisa:

- **Q1**: Atipicidade decisória
- **Q2**: Recorrência de atores (advogados, partes, litigantes)
- **Q3**: Mudança de padrão por período, classe ou colegialidade
- **Q4**: Metodologia e aprofundamento
