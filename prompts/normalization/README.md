# Prompts — normalization

## Objetivo

Armazenar prompts voltados à padronização de campos, nomes e entidades.

## Exemplos de tarefas

- propor regra de normalização de datas;
- descrever como preservar bruto e derivado;
- definir política para nomes de advogados;
- documentar campos incertos;
- estruturar dicionário de dados.

## Regra

Toda proposta de normalização deve preservar o valor bruto e registrar versão da regra.
