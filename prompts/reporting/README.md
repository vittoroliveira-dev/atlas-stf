# Prompts — reporting

## Objetivo

Armazenar prompts usados para gerar relatórios técnicos e executivos a partir das saídas já validadas.

## Exemplos de tarefas

- redigir relatório de anomalias;
- sintetizar perfil de ministro;
- montar lista priorizada de casos;
- revisar linguagem para neutralidade metodológica.

## Regra

Relatórios devem sempre mencionar fontes, limites e natureza não conclusiva dos alertas.
