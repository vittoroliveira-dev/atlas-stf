# Prompt — reporting — TEMPLATE

## Como usar este modelo

Copie este arquivo para um prompt de relatório específico. Remova as instruções quando transformar este modelo em prompt final.

## Nome do prompt

Defina um nome curto e orientado ao artefato final.

## Objetivo

Descreva qual relatório, perfil ou síntese será produzido.

## Quando usar

Usar para montar relatórios técnicos, perfis, listas priorizadas e sínteses executivas a partir de saídas já validadas.

## Entradas disponíveis

- relatórios intermediários
- alertas revisados
- métricas consolidadas
- notas metodológicas
- limites conhecidos

## Saída esperada

Registre relatório final, sumário executivo, documento técnico ou outro artefato final.

## Restrições obrigatórias

- mencionar fontes
- mencionar limites
- não transformar hipótese em conclusão
- manter linguagem neutra
- não exagerar achados

## Critérios de validação

- fontes citadas
- limites citados
- escopo respeitado
- linguagem prudente

## Prompt

Objetivo:
Descreva o relatório a ser produzido.

Entradas:
Liste apenas os materiais já validados para compor o relatório.

Saída esperada:
Descreva o documento final, seu público e o nível de detalhe exigido.

Restrições:
1. Não use linguagem acusatória.
2. Separe achados, hipóteses e limitações.
3. Não omita incertezas relevantes.
4. Escreva de forma clara e auditável.
5. Não amplie o escopo do relatório além do recorte fornecido.

Validação:
Explique como verificar se o texto final permaneceu proporcional aos dados disponíveis.
