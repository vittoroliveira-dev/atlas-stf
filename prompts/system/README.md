# Prompts — system

## Objetivo

Armazenar instruções-base e regras permanentes para os agentes do projeto.

## Conteúdo esperado

- prompt-base do agente
- regras de linguagem
- rótulos permitidos e proibidos
- formato padrão de resposta
- restrições metodológicas
- regras permanentes de tamanho máximo para arquivos de código

## Observação

Prompts desta pasta devem ser estáveis e mudar pouco.
