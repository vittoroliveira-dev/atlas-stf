# Prompts — review

## Objetivo

Armazenar prompts voltados à revisão metodológica e humana dos alertas e documentos do projeto.

## Exemplos de tarefas

- revisar coerência de um alerta;
- apontar hipóteses alternativas;
- checar se o grupo comparável está bem definido;
- classificar caso como relevante, falso positivo provável ou inconclusivo.

## Regra

Toda revisão deve distinguir fato, hipótese e inferência.
