# Prompts — ingestion

## Objetivo

Armazenar prompts voltados à catalogação, inventário e ingestão controlada de dados.

## Exemplos de tarefas

- mapear fontes;
- documentar exportações;
- registrar cobertura aparente;
- montar inventário de arquivos;
- propor convenções de versionamento raw.

## Regra

Não pedir parsing jurídico nesta pasta. O foco aqui é origem, integridade e rastreabilidade.
