# Prompt — analytics — TEMPLATE

## Como usar este modelo

Copie este arquivo para um prompt analítico específico e substitua as orientações por conteúdo aplicável ao recorte analisado.

## Nome do prompt

Defina um nome curto e descritivo.

## Objetivo

Descreva qual análise será produzida.

## Quando usar

Usar quando a tarefa envolver baseline, comparabilidade, score, alerta, priorização ou revisão metodológica de análise.

## Entradas disponíveis

- entidades canônicas
- grupos comparáveis
- baselines
- relatórios parciais
- notas de revisão

## Saída esperada

Registre documento final, formato e escopo analítico.

## Restrições obrigatórias

- não concluir favorecimento
- não usar linguagem acusatória
- explicitar baseline
- explicitar grupo comparável
- registrar incerteza e hipóteses alternativas

## Critérios de validação

- baseline claro
- comparabilidade clara
- saída reproduzível
- limitações explícitas

## Prompt

Objetivo:
Descreva a análise específica a ser executada.

Entradas:
Liste apenas as entradas efetivamente autorizadas.

Saída esperada:
Descreva o artefato final e o nível de detalhe necessário.

Restrições:
1. Não produza conclusão acusatória.
2. Use apenas linguagem descritiva, comparativa ou de atipicidade.
3. Todo resultado deve apontar seu baseline.
4. Todo alerta deve apontar seu grupo comparável.
5. Registre limites e risco de falso positivo.

Validação:
Descreva como verificar se a saída atende ao escopo.
