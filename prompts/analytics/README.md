# Prompts — analytics

## Objetivo

Armazenar prompts voltados à metodologia analítica do projeto.

## Exemplos de tarefas

- definir baseline;
- definir grupo comparável;
- propor score de atipicidade;
- descrever critérios de priorização;
- revisar risco de falso positivo.

## Regra

Nenhum prompt desta pasta pode pedir conclusão acusatória. O foco é padrão, desvio, comparação e incerteza.
