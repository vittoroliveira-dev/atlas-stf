# Checklist de auditoria — alertas

## Objetivo

Validar se um alerta analítico possui base suficiente para ser exibido ou priorizado.

## Aplicação

Usar este checklist antes de publicar, reportar ou revisar um alerta.

## Itens obrigatórios

- [ ] O alerta possui identificador.
- [ ] O caso sinalizado está identificado.
- [ ] O grupo comparável está identificado.
- [ ] O baseline está explícito.
- [ ] O padrão esperado foi descrito.
- [ ] O padrão observado foi descrito.
- [ ] O tipo de alerta está classificado.
- [ ] O score do alerta foi registrado.
- [ ] Há resumo de evidência.
- [ ] Há nota de incerteza, quando necessária.
- [ ] O alerta usa linguagem neutra.
- [ ] O alerta não faz acusação automática.

## Critérios de bloqueio

A etapa deve ser interrompida se:
- não houver baseline;
- não houver grupo comparável;
- a explicação do alerta for opaca;
- a linguagem for acusatória.
