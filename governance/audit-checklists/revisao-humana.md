# Checklist de auditoria — verificação externa opcional

## Objetivo

Padronizar uma leitura externa opcional de alertas e casos priorizados, sem transformá-la em etapa obrigatória do sistema.

## Aplicação

Usar este checklist quando alguém quiser verificar externamente um alerta ou processo.

## Itens obrigatórios

- [ ] O alerta foi lido integralmente.
- [ ] O grupo comparável foi compreendido.
- [ ] O baseline foi conferido.
- [ ] O caso parece realmente comparável ao grupo.
- [ ] Há hipótese alternativa plausível registrada.
- [ ] Há necessidade de documento textual complementar?
- [ ] O alerta parece relevante para aprofundamento?
- [ ] O caso deve ser descartado como falso positivo?
- [ ] O caso deve ser marcado como inconclusivo?
- [ ] A conclusão da verificação foi registrada com justificativa.

## Classificações possíveis

- relevante para aprofundamento
- falso positivo provável
- inconclusivo
- depende de documento textual

## Critérios de bloqueio

A verificação não pode ser considerada concluída se:
- não houver justificativa escrita;
- não houver classificação final registrada.
